## 23andMe V5/V5.1 SNP-Coverage

23andMe-Chip-Manifests sind Illumina-proprietär (GSA Custom Add-On). Die SNP-Liste ist im Genome-Standard-Panel `All_SNPs_hg38_ref.tab.gz` (WGSExtract-konsolidiert) bereits weitgehend abgedeckt. 23andMe V6 ist (Stand 2026) nicht freigegeben.
